=========================
How to use 1-my_list.py
=========================

Import class:

    >>> MyList = __import__('1-my_list').MyList

Create empty list:

    >>> my_list = MyList()
    >>> print(my_list)
    []

Check inheritance:

    >>> isinstance(my_list, list)
    True

Append values:

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Print sorted list:

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Original list is not changed:

    >>> print(my_list)
    [1, 4, 2, 3, 5]

Negative numbers:

    >>> my_list2 = MyList()
    >>> my_list2.append(-1)
    >>> my_list2.append(10)
    >>> my_list2.append(0)
    >>> my_list2.print_sorted()
    [-1, 0, 10]

Empty list:

    >>> my_list3 = MyList()
    >>> my_list3.print_sorted()
    []
